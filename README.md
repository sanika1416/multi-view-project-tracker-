# Multi-View Project Tracker

A frontend project management tool built using React + TypeScript.

## Features
- Kanban board (4 columns)
- Task creation, edit, delete
- Custom drag and drop
- Due date with overdue logic
- Search functionality
- 500+ generated tasks

## Tech Stack
- React
- TypeScript
- Vite

## Run Locally

npm install  
npm run dev
