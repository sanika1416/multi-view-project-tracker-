# 🚀 Multi-View Project Tracker

A modern, responsive project management application built with **React + TypeScript**, featuring multiple views and a fully interactive Kanban board.

---

## 🔥 Features

- 🧩 **Kanban Board** with 4 columns: To Do, In Progress, In Review, Done  
- ✏️ **Task Management**: create, edit, delete tasks  
- 🎯 **Custom Drag & Drop** for moving tasks between columns  
- ⏳ **Due Dates & Overdue Logic**: highlights overdue tasks  
- 🔍 **Search Functionality**: filter tasks by title  
- ⚡ **Large Dataset Support**: handles 500+ tasks efficiently  

---

## 🛠 Tech Stack

- **React** – Frontend UI  
- **TypeScript** – Type safety & maintainability  
- **Vite** – Fast build & development  
- **DnD Kit** – Drag-and-drop functionality  
- **CSS / Tailwind (if used)** – Styling & responsive design
